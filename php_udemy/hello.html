Git Commands

To improve your skills using git, always you should create a new branch and write your code. When everything it's ready makes a merge with the master branch.

Git Status

This command is to check how is the current status in your repository.

Command: git status

Git Branch

This command is to check how many branches there are available and to know where are you currently.

Command: git branch

Create a new branch

To create a new branch you should use the next command:

Command git branch + name's branch

Git Checkout

This command is to move throught your branches and commits.

Command: git checkout

If you want to create a new branch and move to it, use the next command:

Command git checkout -b + name's branch

Git Add

This command is to add your new files to the repository's head.

Command: git add

git add. include all new files.

git add + name's files to include only the files that you chose.

Git Commit

This command is to prepare your new files to send at your repository.

Command: git commit

Command: git commit -m "Write here your custom message"

How to merge your branches?

To merge your branches it's necessary to go the master branch, to do that you can use the next command:

Command: git checkout master

After that it's time to make a merge to achievement it, you can use the next command:

Command: git checkout merge + name's branch to merge

Git Push

This command is to send your new files to your repository.

Command: git push

Delete Branches

If you have made all your changes and don't need more of the new branch, it's time to delete it.
To do it, you can use the next command:

Command: git branch -d + name's branch
